Architecture Highlights

The implementation mirrors GLiNER2 base-v1 while staying idiomatic to Rust and Candle internals.

  • DeBERTa-v3 encoder with relative position embeddings
  • markerV0 span representation (Linear + GELU + Linear)
  • Classifier MLP 768 → 1536 → 1 and count head 768 → 1536 → 20
  • HF tokenizer-driven collator with schema + text batching

Pipeline

Each stage is explicit and debuggable, from token collation to thresholded span extraction.
